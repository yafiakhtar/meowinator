A Chrome extension that replaces all YouTube images with adorable random cats!

## Features

- Replaces all YouTube images with random cats from [cataas.com](https://cataas.com/)
- Toggle on/off with a beautiful popup interface
- Remembers your preference across sessions
- Fast and efficient with throttled updates
- Works with YouTube's dynamic content loading

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select the `meowinator` folder
5. Visit YouTube and enjoy the cats!

## Usage

1. Click the extension icon in your Chrome toolbar
2. Toggle the switch to enable/disable cat mode
3. The setting is saved automatically and persists across sessions
4. Refresh YouTube pages to see changes if needed
